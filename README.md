# Medical Test Automation 🏥🤖

Automated completion of online medical tests using AI-powered answer selection. This tool uses Playwright for browser automation and supports multiple LLM providers (OpenAI GPT-4o, Anthropic Claude 3.5 Sonnet, or Grok) for intelligent answer selection.

## Features ✨

- ✅ Automatic form filling with Cyrillic character support
- ✅ AI-powered answer selection using state-of-the-art LLMs
- ✅ Beautiful console progress tracking
- ✅ Automatic screenshot capture of final results
- ✅ Support for 100-question medical tests
- ✅ Configurable LLM provider (OpenAI, Anthropic, or Grok)
- ✅ Debug mode with per-question screenshots
- ✅ Headless and visible browser modes
- ✅ Robust error handling and retry logic

## Prerequisites 📋

- Python 3.11 or higher
- pip (Python package manager)
- Internet connection
- API key for at least one LLM provider
